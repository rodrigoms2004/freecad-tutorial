# Freecad tutorial


Freecad studies 

## Install it in Ubuntu 18.04

It is my OS

```
sudo add-apt-repository ppa:freecad-maintainers/freecad-stable

sudo apt update

sudo apt install freecad
```

To remove it

```
sudo apt remove --auto-remove freecad
```
